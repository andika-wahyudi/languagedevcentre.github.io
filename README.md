# Pixel RPG Island (Starter)

A lightweight HTML5 Canvas + ES Modules pixel RPG framework inspired by top‑down exploration mini‑game collections. Original scaffold, no proprietary assets.

## Getting Started
Serve the folder via any static server (required for ES modules):
```
python -m http.server 8080
# or
npx serve .
```
Visit http://localhost:8080

## Controls
- Arrow keys / WASD: Move
- Space: Interact
- M: Mini-game overlay

## Structure
- index.html
- style.css
- game.js (entry point)
- engine/ (modules)
- assets/ (sprite placeholders)

## Extend
Add real art, more quests, more mini-games, persistence, audio.
